The GUI sbx deserved

From your first sandbox to a fleet of AI agents — den makes every step feel effortless.

🏖️

Visual Dashboard

See every sandbox at a glance. Status, resource usage, agent activity — one clean view, nothing buried in logs.

⚡

One-Click Launch

Spin up an isolated microVM in seconds. Pick your environment, configure resources, and hit go.

👁️

Live Agent Monitor

Watch your AI agents work in real time. Streaming logs, file events, and container activity — all surfaced clearly.

📂

File Explorer

Browse the sandbox filesystem from your desktop. Open, inspect, and copy files without manual volume mounts.

Environment Builder

Craft reusable sandbox templates visually. Layer tools, packages, and config — then save for next time.

🔒

Secure by Default

Every sandbox gets its own Docker daemon, filesystem, and network. Your host stays completely untouched.

Simple as sand

Up and running in minutes

den sits on top of your existing sbx setup. If you've used Docker Sandboxes, you're already halfway there.

1

Install den

Download the app, drop it in Applications. No Homebrew, no npm install, no setup wizard.

2

Connect to sbx

Sign in with your Docker account. den picks up your sbx credentials automatically.

3

Create a sandbox

Pick a base image, tweak resources if you like, and launch. Your isolated microVM is ready in seconds.

4

Run your agents

Launch Claude or any AI agent into the sandbox. Watch it build, explore, and ship from the dashboard.
