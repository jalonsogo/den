# sbx logout

**Description** — Stop all running sandboxes and sign out of Docker
**Usage** — `sbx logout [flags]`

## Options

| Option | Default | Description |
| --- | --- | --- |
| `-y, --yes` |  | Skip confirmation prompt |

## Global options

| Option | Default | Description |
| --- | --- | --- |
| `-D, --debug` |  | Enable debug logging |
